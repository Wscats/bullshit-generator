# Settings

你可以在 `settings.json` 里面更改以下配置来切换语言：

```json
{
  "bullshift.language": "EN"
}
```

或者再设置搜索 `@ext:wscats.bullshift`， 切换生成的语言，EN 为英文，CN 为中文(default)。
